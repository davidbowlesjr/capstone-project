# Python Virtual Environments

## Virtual Environment

### To create the environment

```
python -m venv .venv
```

### To activate the virtual environment

- Source Method

```
source venv/Scripts/activate
```

- Powershell
    
    ```
    \Scripts\Activate.ps1
    ```
    
- CMD
    
    ```
    \Scripts\activate.bat
    ```
    

### To Deactivate

'''
deactivate
'''
