# How to run backend
## Set Up Virtual Environment ##
### To create
- python -m venv .venv

## Activate the Virtual Environment ##
- Powershell
    - <venv>\Scripts\Activate.ps1
- CMD
    - .venv\Scripts\activate.bat

## Install Requirements ##
 - pip install -r requirements.txt

## Run App ## 
- python app.py
